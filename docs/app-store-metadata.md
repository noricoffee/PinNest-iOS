## 説明文（4000文字以内）

```
pinNest は、URLや画像・動画・PDF・テキストなど、あらゆるコンテンツをデバイス内に保存・管理できるコンテンツマネージャーです。

【5種類のコンテンツに対応】
・URL：ページのサムネイル・タイトルを自動取得して見やすく表示
・画像・動画：フォトライブラリから選択してそのまま保存
・PDF：ファイルをインポートして一元管理
・テキスト：メモや引用文を直接入力して保存

【Share Extension でかんたん保存】
Safari や他のアプリの共有メニューから pinNest を選ぶだけで、すぐにピン留めできます。その場でタイトルやメモを追加することも可能です。

【タグ & キーワード検索】
保存したコンテンツにタグを付けて分類。キーワード検索とタグフィルターを組み合わせることで、大量のピンの中から目的のものをすばやく見つけられます。

【完全オフライン動作】
すべてのデータはデバイス内にのみ保存されます。クラウド同期も外部サーバーへの送信もなく、プライバシーを守りながら安心して使えます。

【ダークモード対応】
システム設定に合わせた自動切り替えに加え、アプリ内でライト／ダークを手動設定することもできます。

【アクセシビリティ対応】
Dynamic Type・VoiceOver に対応し、文字サイズの変更やスクリーンリーダーでも快適に操作できます。

「後で読もう」と思ったページ、メモしておきたいテキスト、大切な画像やPDF——pinNest でまとめてピン留めしておきましょう。
```
